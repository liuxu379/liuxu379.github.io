## 前言

百度还能用吗？搜索出来的东西几乎全是 Ctrl + C 和 Ctrl + V 的产物，估计自己都没测试就发出来。



## 解决方法

编辑 `/etc/ssh/sshd_config`

将 **32** 行的 **PermitRootLogin** 参数 改为 **yes**

``` 
32 #PermitRootLogin prohibit-password
33 PermitRootLogin yes
```

重启 SSH 服务就可以了

`sudo service ssh restart`
