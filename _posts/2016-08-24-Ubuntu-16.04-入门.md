## 前言

2014年买的笔记本，重装了 N 次系统了，尝试了很多 Linux 发行版本，最终选择了 Ubuntu，简单说一下它的特点：

- **使用人数多**
- **软件资源丰富**
- **界面美观**
- **驱动支持较好**

***

#### 安装软件

- **APT 方式安装**

  `sudo apt-get install 软件名称（openjdk-8-jdk、rapidsvn、filezilla等）`

- **安装 DEB 文件**

  `sudo dpkg -i -f 文件名称`

  > 安装 DEB 格式的软件，最常碰到的问题就是缺少依赖，可以使用`sudo apt-get install -f`命令来补全依赖，然后再次安装软件。
  >
  > 这里添加了`-f`参数，可以直接进行补全的操作。

***

#### 安装主题、图标和字体

```
sudo apt-get install unity-tweak-tool
sudo add-apt-repository ppa:noobslab/themes
sudo add-apt-repository ppa:numix/ppa
sudo apt-get update
sudo apt-get install flatabulous-theme
sudo apt-get install fonts-wqy-microhei
sudo apt-get install numix-icon-theme-circle
```

***
